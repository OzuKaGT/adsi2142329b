- Para el funcionamiento del portafolio se debe ingresar primero la base de datos. El archivo SQL se encuentra en la carpeta Modelo - Se le debe nombrar a la base de datos "portafolio".

- Despues de haberse instalado la base de datos, para agregar peticiones, se debe ir a la parte de abajo del Index.html donde se encuentra un formulario para ingresar datos.

- Si se quieren ver los datos de la base de datos, en la barra de navegacion se debe dar click al boton "Mostrar Peticiones"

- En esa pagina se encuentran los botones "editar" y "eliminar" en frente de cada dato registrado. 

- El resto de opciones en la barra de navegacion son "Inicio", "Trabajos" y "Manuales".

- El inicio sirve como boton para ir a la pagina principal.

- Trabajos es una barra desplegable el cual nos da opciones para ver diferentes trabajos en JavaScript.

- Manuales es tambien una barra desplegable que nos da la opcion de ver como se hace un manual de usuario y uno tecnico.

Y eso seria todo principalmente.
