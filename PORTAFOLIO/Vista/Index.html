Lenguajes de Programacion


Los siguientes lenguajes de programacion son los que más domino y de los que principalmente esta hecha está pagina.




HTML5

Es el lenguaje basico, el cual crea la estructura base de las paginas web. Esta pagina fue trabajada con este lenguaje.

CSS3

Es el lenguaje basico sobre la interfaz y el estilo, este se crea usando estructuras donde se determinan todo tipo de aspectos visuales. Está pagina está trabajada con este lenguaje de programacion.

PHP

Es el lenguaje de programacion por el cual se realiza la conexion a una base de datos y por medio de la cual se interactua. Está pagina utiliza lenguaje PHP para su funcionamiento.

JavaScript

Es un lenguaje de programacion el cual va orientado a crear una pagina web mucho maás dinamica, con acciones diferentes, ventanas emergentes, ventanas desplegables, etc. Está pagina tiene implementado este lenguaje.

Bootstrap

Es un sistema de plantillas web que hace uso de lenguaje CSS y JavaScript para crear diferentes implementos como barras de navegacion o tablas. Está pagina usa codigo Bootstrap.
